Installation Guide

Important

Warnings

  • Installing may erase your whole disk if misconfigured.
  • Manual Installation is for advanced users only. GUI installation is the best for most users.

    • Requirements

    • USB drive (8GB+)
    • Electricity (optional)

    GUI Install

    Download ISO

    Download the latest Snares Linux ISO from the website.

    Flash the USB

    Use a tool like Rufus (Windows 10/11) or Balena Ethcer (Linux/macOS) to flash the iso onto the USB drive.

    Boot the USB

    Access the devices boot manager and select the drive to boot it. When met with a black and white menu choose the first option to boot the installation enviroment.

    Install Wizard

    Find the Install Wizard App, it is often find in the dock and if not look for it in applications. Open the app and follow the instructions to complete the install.

    Manual Install

    Partitioning and Formatting

    Partition the drive using any of the already installed Disk Management Tools.

  • fdisk

  • cfdisk

    • Use lsblk to locate the drive you want to partition. In this guide we will say the drive is /dev/sda.
    Execute fdisk/cfdisk with the drive as an argument. Partition the drive to at least have a EFI partition(512M+) and a root partition(3G+)

    Now you need to format the partitions, start with the EFI partition: mkfs.vfat /dev/sda1, then continue with the root: mkfs.ext4 /dev/sda2. If you created a swap partition format that too: mkswap /dev/sda3
    Now your drive is partitioned and formatted and you can continue with mounting the partitions.

    Mounting

    Mount the partitions.

    Installing the base system

    Snares Linux uses Void's xbps for package management and installation. To install the base system execute: xbps-install -S -r /mnt snares-base . Once that completes you can continue with the installation of kernel and bootloader.

    Installing kernel and bootloader

    Now with the base system mounted on /mnt can we enter that system with the snares-chroot command. To enter using snares-chroot use that command with /mnt as an argument: snares-chroot /mnt . Now inside the system you do the same as with the base system, we install the kernel and bootloader with xbps.
    We can begin with installing the most important piece, the kernel. Install the kernel like this: xbps-install -S linux .

    Now we install the bootloader, Snares Linux uses grub as the standard bootloader but feel free to install any other. Get the grub install command like this: xbps-install -S grub .
    Then execute the grub installer on your drive: grub-install /dev/sda .
    Grub is now installed and now only needs a config which can be automatically generated like this: grub-mkconfig -o /boot/grub/grub.cfg .

    Configuring system before reboot

    Before rebooting we recommend you do all of these steps:

  • Change root password: passwd
  • Add a new user: useradd -m USERNAME
  • Change password of new user: passwd USERNAME
  • Changing hostname

    • Now your system is ready for its first boot.
