Installation Guide

Important

Warnings

  • Installing an operating system may erase data if the wrong drive or partition is selected.
  • Always back up important files before installing Snares Linux.
  • The GUI installer is recommended for most users.
  • The manual installation method assumes basic familiarity with Linux command-line tools.

Requirements

  • USB flash drive (8 GB or larger)
  • A computer capable of booting from USB
  • Internet connection (recommended)

GUI Installation

1. Download the ISO

Download the latest Snares Linux ISO image from the official download page.

2. Create Installation Media

Write the ISO image to a USB flash drive using one of the following tools:

  • Rufus (Windows)
  • Balena Etcher (Windows, Linux, macOS)
  • Ventoy

After the flashing process completes, safely eject the USB drive.

3. Boot the USB

Insert the USB drive into the target computer and reboot.

Open your system's boot menu (commonly F12, F11, ESC, or DEL depending on your hardware) and select the USB device.

When the Snares Linux boot menu appears, select the first option to start the live environment.

4. Run the Installer

Once the desktop environment has loaded, open the Install Wizard.

The installer is usually available from the dock. If it is not present, open the application menu and search for "Install Wizard".

Follow the on-screen instructions to configure storage, create users, and install the system.

Manual Installation

1. Partitioning and Formatting

First, identify the target drive: 

lsblk

In this guide we will assume the target drive is: 

/dev/sda

Use your preferred partitioning tool:

  • fdisk
  • cfdisk

Example: 

cfdisk /dev/sda

Create at minimum:

  • EFI System Partition (512 MB or larger)
  • Root partition (10 GB or larger recommended)
  • Optional swap partition

Format the partitions: 


mkfs.vfat -F32 /dev/sda1
mkfs.ext4 /dev/sda2
mkswap /dev/sda3

Skip the swap command if you did not create a swap partition.

2. Mounting Partitions

Mount the root partition: 

mount /dev/sda2 /mnt

Create the boot directory: 

mkdir -p /mnt/boot

Mount the EFI partition: 

mount /dev/sda1 /mnt/boot

If you created a swap partition, enable it: 

swapon /dev/sda3

3. Installing the Base System

Snares Linux uses the XBPS package manager.

Install the base system to the mounted filesystem: 

xbps-install -S -r /mnt snares-base

Once installation completes, continue with the kernel and bootloader.

4. Installing the Kernel and Bootloader

Enter the newly installed system: 

snares-chroot /mnt

Install the Linux kernel: 

xbps-install -S linux

Install GRUB: 

xbps-install -S grub

Install the bootloader: 

grub-install /dev/sda

Generate a GRUB configuration: 

grub-mkconfig -o /boot/grub/grub.cfg

Snares Linux uses GRUB by default, but advanced users may choose an alternative bootloader.

5. Final Configuration

Before rebooting, perform the following recommended steps:

Set the Root Password

passwd

Create a User

useradd -m username

Set the User Password

passwd username

Set the Hostname

echo "snares" > /etc/hostname

Exit the chroot environment: 

exit

Unmount all partitions: 

umount -R /mnt

Reboot the system: 

reboot

Remove the installation USB when prompted. Your Snares Linux installation should now boot normally.
